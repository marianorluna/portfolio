Discover how to transform your BIM workflow by bringing artificial intelligence directly into your models. Using the **MCP** (Model Context Protocol), **pyRevit**, and **Cursor**, you can talk to your model, audit parameters, automate modeling, and write scripts on the fly without leaving the chat.

This guide walks you through everything from installation to running your first AI prompt interacting with Revit in real time. It's written for people who aren't used to these tools: every step explains what to do and why.

## Prerequisites

Before you start, make sure you have this ready:

**1. Global Python environment.** The MCP protocol needs Python as an intermediary server. Once installed, open your terminal (CMD or PowerShell) and install the base libraries:

```bash
pip install mcp httpx
```

**2. Autodesk Revit 2027.** You need this exact version installed.

## Step-by-step guide

<Step number={1} title="Install pyRevit">

Download the pyRevit installer from its official GitHub repository. The first time you open Revit after installing it, make sure to tell it to **always load** this add-in.

<Callout variant="tip">
Use the "Admin / All Users" installer to avoid permission issues.
</Callout>

</Step>

<Step number={2} title="Configure Routes in pyRevit">

Go to the **pyRevit** tab → **Settings**.

- In the left panel, look for **Routes**.
- Turn on the **Enable Routes Server (Beta)** switch.
- Check the **pyrevit-core/** box.
- Click **Save Settings and Reload** to apply the changes.

</Step>

<Step number={3} title="Install the MCP extension">

Go to **pyRevit → Extensions**. Find and install the extension named `mcp-server-for-revit-python`. Once installed, reload Revit or pyRevit again.

</Step>

<Step number={4} title="Verify the Revit connection">

<Callout variant="critical">
**Critical!** Open a project or family in Revit. The server won't work while you're on the Home screen.
</Callout>

Open your browser and go to:

```text
http://localhost:48884/revit_mcp/status/
```

If everything is correct, you should see a JSON response like this:

```json
{
  "status": "healthy",
  "api_name": "revit_mcp",
  "revit_available": true
}
```

</Step>

<Step number={5} title="Configure Cursor">

Download Cursor from its official website if you don't have it yet. Inside Cursor, open **Settings → Features → MCP** and click **+ Add New MCP Server**. Configure it as "command" with the following JSON (replace `your_username` with your actual Windows username):

```json
{
  "mcp-server-for-revit-python": {
    "command": "python",
    "args": [
      "C:\\Users\\your_username\\AppData\\Roaming\\pyRevit\\Extensions\\mcp-server-for-revit-python.extension\\main.py"
    ]
  }
}
```

Check that a **green dot** appears next to the server name: it means Cursor has successfully handshaken with pyRevit.

</Step>

<Step number={6} title="First test!">

Open the Cursor chat and send this prompt to check the connection:

<PromptExample label="Example prompt">
"Check the status of Revit 2027 and tell me the name of the active document and its project units."
</PromptExample>

You'll see Cursor automatically invoke the `get_revit_status` tool, pull information from your model, and describe it back to you in plain language.

</Step>

## Use cases and tools

The MCP server exposes around 20 tools that Cursor's AI can use autonomously based on what you ask it.

### Data auditing and extraction

Extract valuable information without creating temporary schedules.

Key tools: `get_current_view_elements`, `list_category_parameters`, `get_revit_model_info`.

<PromptExample label="Example prompt">
"List all the walls visible in the current view, group them by family type, and tell me the total 'Volume' parameter value for each group."
</PromptExample>

### Modeling automation

Delegate repetitive element placement or project structuring.

Key tools: `place_family`, `list_levels`, `list_families`.

<PromptExample label="Example prompt">
"Find the 'Meeting Table' family in the model and place one on Level 1, exactly at coordinates X:10, Y:5, Z:0."
</PromptExample>

### Graphics management and quality control

Apply visual overrides instantly to highlight errors or classify elements.

Key tools: `color_splash`, `clear_colors`, `list_revit_views`.

<PromptExample label="Example prompt">
"Paint all elements in the 'Doors' category bright red in this view so I can find them quickly. Once I'm done reviewing them, clear the overrides."
</PromptExample>

### The ultimate wildcard: direct code

If there's no predefined tool for what you need, Cursor can write and run a Revit API script on the fly.

Key tools: `execute_revit_code`.

<PromptExample label="Example prompt">
"Write and run a Python script in Revit that selects all the walls in the project and copies the text from the 'Type Mark' parameter into the 'Comments' parameter."
</PromptExample>
